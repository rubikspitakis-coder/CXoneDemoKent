Cognigy API Endpoints

Configure these endpoints in your Cognigy flow to synchronise voice interactions with this dashboard in real time.

GET /api/member/:phone

Look up a member by caller ID. Returns full member record as JSON.

PATCH /api/member/:phone

Update status, CPD hours, or notes. Triggers a real-time dashboard flash.

POST /api/reset

Resets all member data to the initial seed state for demo purposes.
